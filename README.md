CircularLayout - кастомный ViewGroup, в который можно добавлять другое контейнеры. 
Если суммарная ширина добавленных контейнеров будет больше, 
чем ширина CircularLayout, то скроллинг будет зациклен по кругу.
Так же внизу будет отображаться текущий прогресс перемотки.
